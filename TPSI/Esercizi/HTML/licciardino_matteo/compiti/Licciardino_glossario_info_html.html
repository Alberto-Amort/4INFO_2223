Glossario di informatica
VOCE DESCRIZIONE FONTE
;first-letter   Pseudo elemento (X)HTML, è
possibile formattare la prima
lettere di qualunque elemento
contenente del testo. 		  Lezione_30 (CSS-
ClassiRegole)
:after   Pseudo elemento (X)HTML
complementare a :before.
L'unica differenza è
che il contenuto viene inserito dopo
l'elemento specificato nel selettore. 		  Lezione_30 (CSS-
ClassiRegole)
:before   Pseudo elemento (X)HTML che
consente di inserire nel documento
l'elemento specificato nel selettore.		   Lezione_30 (CSS-
ClassiRegole)
:first-line   Pseudo elemento (X)HTML,
imposta lo stile della prima riga
di un elemento contenente del testo.		   Lezione_30 (CSS-
ClassiRegole)
@-import   Usandto per collegare un foglio
di stile esterno al documento. 		  Lezione_30 (CSS-
ClassiRegole)
@-rules   Alternativa all'elemento <link>
per collegare fogli esterni. 		  Lezione_30 (CSS-
ClassiRegole)
@charset   Serve a specificare l'impostazione
relativa alla codifica dei caratteri
di un documento. 		  Lezione_30 (CSS-
ClassiRegole)
@font-face   Usata per descrivere
un font usato nel documento. 		  Lezione_30 (CSS-
ClassiRegole)
<audio></audio>   Consente di arricchire le pagine
HTML di un sottofondo sonoro. 		  Lez_26 (Validazione)
<base>   Usato per definire l'URL
di base della pagina. 		  Lez_28 (StrutturaPagina)
<canvas>   Tag utilizzato per includere
grafica attraverso codice Javascript. 		  Lez_21 (Multimedia)
<img>   Tag che consente di inserire
immagini in un documento HTML. 		  Lez_26 (Validazione)
<isindex>   Modo per creare elementi
simili alle caselle di testo. 		  Lez_28 (StrutturaPagina)
<link>   Contiene informazioni su   Lez_28 (StrutturaPagina)
